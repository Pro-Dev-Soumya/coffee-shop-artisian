# ☕ Soumya's Coffee Shop Website

A simple and elegant coffee shop website built using **HTML, CSS, and JavaScript**. This project showcases a modern, responsive design suitable for small businesses like cafés and restaurants.

---

## 📌 Project Overview

Soumya's Coffee Shop website is designed to provide customers with an engaging online experience. It includes sections like menu display, about us, contact information, and interactive UI elements.

---

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Attractive homepage with banner/hero section
* Menu section with categorized items
* About section describing the coffee shop
* Contact form with basic validation
* Smooth scrolling and simple animations using JavaScript
* Clean and modern UI with CSS styling

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout (Flexbox/Grid)
* **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior

## ⚙️ Installation & Setup

1. Clone the repository:

   ```
   git clone https://github.com/your-username/coffee-shop.git
   ```

2. Navigate to the project folder:

   ```
   cd coffee-shop
   ```

3. Open `index.html` in your browser.
